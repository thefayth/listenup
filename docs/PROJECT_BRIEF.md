# Project Brief

## What This Is

ListenUp is a transcription and caption platform for clean, searchable media workflows. It supports uploaded owned media, direct media URLs, pasted captions, saved transcript history, and TXT/SRT/VTT export workflows.

## Why It Matters

Raw transcription is rarely the final product. ListenUp is designed around the work that comes after transcription: editing, searching, saving, and exporting text that can be used by creators, media teams, advocates, researchers, and operators.

## Product Shape

- Studio Console workspace.
- Account-based transcript history.
- Safety-first media intake.
- Guided social-link handling instead of brittle scraping.
- Export-ready transcript output.
- Revenue-ready sponsor and billing surfaces.

## Public Surface

This repository documents the project, visuals, status, ownership, and public/private boundary. The private source code and operational systems remain protected.
