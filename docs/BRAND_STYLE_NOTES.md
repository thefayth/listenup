# Brand Style Notes

## Direction

ListenUp uses the Studio Console visual direction: crisp, useful, serious, and media-workstation oriented.

## Palette

- Canvas: `#f6f8fb`
- Surface: `#ffffff`
- Ink: `#101828`
- Muted text: `#667085`
- Primary cobalt: `#2454ff`
- Success green: `#0f9f6e`
- Warning amber: `#c77900`
- Line: `#d9e1ec`

## Motifs

- Waveform bars.
- Transcript rows.
- Timecode rhythm.
- Clean account and history panels.
- Public/private boundary diagrams.

## Avoid

- Generic SaaS gradients.
- Soft cream landing-page tropes.
- Unclear stock imagery.
- Visuals that imply the private engine is open-source.
