# Public / Private Boundary

## Public In This Repository

- Project description and purpose.
- Product screenshots selected for public use.
- Workflow diagrams.
- Status and roadmap.
- WordPress page draft.
- Ownership, security, trademark, and commercial-use policies.
- Receipts for the public export package.

## Private And Not Included

- Application source code.
- Prompts, agent instructions, and private workflows.
- Credentials, tokens, API keys, cookies, and secret values.
- User accounts, sessions, transcripts, billing records, and customer data.
- Deployment systems, server configs, backups, logs, and infrastructure details.
- Legal/admin records, family records, medical records, benefits records, and unpublished creative works.

## Operating Principle

Public surface. Private engine. Receipts always.
