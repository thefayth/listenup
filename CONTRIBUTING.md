# Contributing

ListenUp is not accepting public source contributions through this repository.

This repository is a public project surface, not the full engineering workspace. Public issues or comments may be used for high-level product interest, partnership inquiries, or public documentation corrections. Do not submit source code, secrets, private data, copied transcripts, customer records, or sensitive operational details.

Commercial, partnership, sponsorship, and licensing conversations should go through Faith Cheltenham's official contact paths.
