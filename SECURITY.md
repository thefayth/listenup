# Security

This public repository does not include the production source code, credentials, operational infrastructure, customer data, private workflows, or deployment configuration for ListenUp.

## Reporting

If you believe you have found a security issue related to a public ListenUp surface, contact Faith Cheltenham through the official FaithCheltenham.com contact path.

Do not post credentials, private data, exploit details, or sensitive operational information in public issues.

## Boundary

Security-sensitive materials that remain private include:

- API keys, tokens, cookies, credentials, and secrets.
- Source code and private implementation details.
- Deployment configuration, server paths, backups, logs, and database files.
- User data, transcripts, account records, and billing records.
- Agent instructions, prompts, private workflows, and recovery files.
